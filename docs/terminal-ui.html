Terminal UI Guide

Master the keyboard-driven interface for fast cluster navigation

Overview

The KubeGraf Terminal UI provides a fast, keyboard-driven interface for exploring and managing your Kubernetes cluster. It's designed for efficiency, allowing you to navigate resources without leaving your terminal.

Launching the TUI

# Default namespace kubegraf # Specific namespace kubegraf production # View version kubegraf --version

Interface Layout

The terminal interface consists of:

  • Tab Bar - Navigate between resource types (Pods, Deployments, Services, etc.)
  • Resource Table - Main view showing resources with status icons
  • Status Bar - Current namespace, cluster info, and metrics
  • Event Bar - Recent Kubernetes events
  • Help Bar - Quick reference for keyboard shortcuts

Resource Tabs

Pods

View pod status, IP, restarts, age, CPU/memory usage

Deployments

Replica status, available pods, conditions

StatefulSets

Ordered pod management with persistent storage

DaemonSets

Pods running on all (or selected) nodes

Services

Service type, cluster IP, external IP, ports

Ingresses

HTTP routing rules and backends

ConfigMaps

Configuration data for applications

Nodes

Cluster node status and capacity

Canvas Graph (Terminal)

Press i to launch the interactive canvas graph directly in your terminal. This shows resource relationships in ASCII art with keyboard navigation:

  • Arrow keys - Pan the view
  • + / - - Zoom in/out
  • q - Exit graph view

Tips

  • Use r frequently to refresh data
  • The vim-style j/k navigation works throughout
  • Delete requires confirmation - press y to confirm
  • Shell access (s) only works on running pods
