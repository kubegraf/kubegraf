Topology Visualization

Interactive force-directed graphs showing resource relationships

Overview

KubeGraf provides two ways to visualize your cluster topology:

  • Terminal Canvas Graph - ASCII-based visualization in your terminal
  • Web D3.js Graph - Interactive browser-based visualization

Web Topology View

Access the interactive topology at:

  • http://localhost:8080/topology (when running web dashboard)
  • Click "Topology" in the web dashboard navigation

Interactions

  • Drag - Move nodes to rearrange the layout
  • Scroll/Pinch - Zoom in and out
  • Hover - See node details and highlight connections
  • Click - Select a node to focus on its relationships

Relationship Lines

The graph shows connections between resources:

  • Deployment → Pods - Pods managed by a deployment
  • Service → Pods - Pods selected by a service
  • Ingress → Service - Backend services for ingress routes
  • Pod → ConfigMap/Secret - Mounted configurations

Terminal Canvas Graph

Press i in the terminal UI to launch the ASCII canvas graph.

Controls

  • Arrow keys - Pan the view
  • + / - - Zoom in/out
  • r - Reset view
  • q / Esc - Exit graph

Export Options

From the terminal UI, press g to export the topology as an HTML file that opens in your browser. This creates a standalone D3.js visualization you can share.
