Getting Started

Everything you need to install and start using KubeGraf

Prerequisites

Before installing KubeGraf, ensure you have:

  • A Kubernetes cluster (local or remote)
  • kubectl configured with access to your cluster
  • Go 1.21+ (for source installation)

Installation

Option 1: Go Install (Recommended)

The easiest way to install KubeGraf:

go install github.com/kubegraf/kubegraf@latest

Option 2: Homebrew (macOS)

brew install kubegraf/tap/kubegraf

Option 3: Binary Download

Download the pre-built binary for your platform:

# Linux (amd64) curl -L https://github.com/kubegraf/kubegraf/releases/latest/download/kubegraf-linux-amd64 -o kubegraf # macOS (arm64 - Apple Silicon) curl -L https://github.com/kubegraf/kubegraf/releases/latest/download/kubegraf-darwin-arm64 -o kubegraf # Make executable chmod +x kubegraf sudo mv kubegraf /usr/local/bin/

Option 4: Build from Source

git clone https://github.com/kubegraf/kubegraf.git cd kubegraf go build -o kubegraf ./kubegraf

Quick Start

Terminal UI

Launch the interactive terminal interface:

# Default namespace kubegraf # Specific namespace kubegraf production

Web Dashboard

Launch the web-based dashboard:

# Default port (8080) kubegraf --web # Custom port kubegraf --web --port=3000

Then open http://localhost:8080 in your browser.

Tip: The web dashboard will start even if your cluster is not connected, showing a connection error banner. This allows you to see the UI and diagnose connection issues.
