# Security Policy

## Supported Versions

We actively support security updates for the latest stable release of KubeGraf.

## Reporting a Vulnerability

If you discover a security vulnerability in KubeGraf, please report it responsibly.

### How to Report

**Email:** contact@kubegraf.io  
**Subject:** `Security: <short title>`

Please include:
- A clear description of the vulnerability
- Steps to reproduce the issue
- The affected KubeGraf version
- Potential impact assessment
- Any suggested mitigations

### What to Expect

We will acknowledge receipt of your report and respond within a reasonable timeframe. We appreciate your patience as we investigate and address security issues.

### Responsible Disclosure

We ask that you:
- Do not publicly disclose the vulnerability until we've had a chance to address it
- Allow us time to investigate and release a fix
- Act in good faith to avoid privacy violations, data destruction, or service interruption

Thank you for helping keep KubeGraf and its users safe!
