# Security Policy

## Supported Versions

| Version | Security support |
|---|---|
| Latest stable minor | Full — patches shipped within SLA |
| Previous stable minor | Security patches only, for 90 days |
| Older versions | Not supported — please upgrade |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities via [GitHub Security Advisories](https://github.com/kubegraf/kubegraf/security/advisories/new).

We will acknowledge your report within **24 hours** and provide an initial assessment within **72 hours**.

## Response SLAs

| Severity (CVSS) | Acknowledgement | Patch target |
|---|---|---|
| Critical (9.0+) | 4 hours | 24 hours |
| High (7.0–8.9) | 24 hours | 72 hours |
| Medium (4.0–6.9) | 72 hours | Next scheduled release |
| Low | 7 days | Next minor release |

Patches are shipped as PATCH releases (`v1.x.y`). A public advisory is published after the patch is available.
